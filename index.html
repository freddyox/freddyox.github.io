Overview of Projects

Programming based projects during my years as a physics graduate student.

My Ph.D. analysis required computational and programming skills that I wanted to acquire. In order to quickly learn, I started working on often physics-inspired projects during my spare time, and eventualy developed new passion. The purpose of this site is to organize all my side-projects for my own satisfaction, and for any one else that may find the projects interesting. While a deeper analysis may be performed on most problems, I consider a problem complete based on boredom and time considerations. The programming language of choice is typically C++ (not that it matters), and I make heavy use of CERN's scientific framework ROOT and the multimedia library SFML. Many visualization algorithms and tools are developed from scratch using barebone libraries, e.g. SFML provides powerful low-level tools giving the user significant flexibility.
